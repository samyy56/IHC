Exemplo 9 CSS - Display

Display in-line

O display in-line é utilizado para fazer com que o os elementos HTML sejam disponibilizados na mesma linha. Quando utilizado, o display in-line, não inicia uma nova linha apenas ocupa o espaço necessário para o seu conteúdo.

Display bock

Elementos com display bloc ocupam toda largura disponível e sempre começam em uma nova linha. São exemplos com o uso do display block as divs paragráfos, as seções h1 ao h6.
