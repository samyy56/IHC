Exemplo 1 CSS - Seletores de Tipo

Um seletor de tipo é aquele que representa uma tag HTML e é utlizado em uma regra CSS para indicar a propriedade e valor que aquele tipo HTML deverá assumir após a aplicacação da regra CSS.

Por exemplo: Esses parágrafos vão ter estilos próprios, aplicados a tag "p".
