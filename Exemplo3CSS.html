Exemplo 3 CSS - Seletores de Classe

Seletor de Classe

Os seletores de Classe são aqueles utilizados para aplicar regras css a classes específicas. Eles fornecem uma maneira de selecionar elementos com base em uma classe que foi definida em seu html.

Sua sintaxe começa sempre com .nome seletor {propridade:valor}, abaixo exemplificamos este tipo de seletor.
