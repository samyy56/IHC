Exemplo 2 CSS - Seletores de ID

Um seletor de ID é aquele que representa um elemento HTML que possui um ID(único) onde a regra CSS será aplicada. O seletor de ID sempre começa com um "." e assim como o seletor de tipo estará associado a uma regra CSS.
