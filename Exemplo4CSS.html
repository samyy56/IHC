Exemplo 4 CSS - Vinculação de Arquivos CSS

Vinculação Externa

A vinculação de arquivos CSS de forma externa é aquela onde as regras CSS são criadas em um arquivo externo e vinculadas ao documento HTML por meio de referências.

Esse tipo de vinculação geralmente é realizado tag link. Ou ainda, usnado o @import, que é uma outra forma de linkar um arquivo CSS ao documento HTML.

Vinculação incorporada

A vinculação de arquivos CSS pode ser realizada diratemente na página, nos denominamos esse tipo de vinculação como sendo a incorporada. Essa forma não é a mais recomendada, pois gera um trabalho maior na hora de realizarmos manutenções.

Esse tipo de vinculação é criado por meio da tag style, geralmente colocando no cabeçalho da página HTML.

Vinculação inline

Esse tipo de vinculação é usado para dar estilo diretamente a um elemento HTML. Não é muito recomendado, pois dificulta a sua manutenção.

Para utilizá-lo você deve usar o atributo style diretamente no elemnto HTML.
